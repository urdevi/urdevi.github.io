This page was created to present the Sandboxed Netwrok which was required for the Networking and Security Practices. Based on the requirements which have been provided there was supposed to be three VM's. The VM's that had to be deployed was Ubuntu Desktop 22.04, Ubuntu Server 22.04 which should act as a Gateway and provide Internet Access, and lastly Bitnami application server in which case i choosed OpenCart.
